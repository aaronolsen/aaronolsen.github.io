3D 5-bar Linkage

To run the example below, ensure that you have R installed on your system as well as the R packages linkR and svgViewR. Then, copy and paste the following code into the R console. 

# Load the linkR and svgViewR packages
library(linkR)
library(svgViewR)

# Set the joint coordinates
joints <- matrix(c(0,0.5,0.5, 0.5,1.1,0.5, 1,0.7,0, 1.5,1,0, 1.5,0.5,0), 
	nrow=5, ncol=3, byrow=TRUE)

# Set the joint constraint vectors
joints.cvec <- list(c(0.2,0,1), NA, c(1,0,0.2), NA, c(0.1,0,1))

# Define the linkage
linkage <- defineLinkage(joints=joints, joints.type=c("R", "U", "L", "U", "R"),
	joints.cvec=joints.cvec)

# Animate the linkage
link_anim <- animateLinkage(linkage, t=seq(0, 0.2, length=50))

# Draw the linkage using svgViewR
drawLinkage(link_anim, file="3d_5-bar.html", animate=TRUE, animate.reverse=TRUE)

This will create an interactive visualization as an '.html' file that can be opened in any compatible web browser. See svgViewR Interactive Commands for instructions on how to interact with svgViewR animations.
