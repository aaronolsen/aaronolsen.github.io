Segmenting elements from a CT scan

    Installing 3DSlicer

  1. Download 3DSlicer.

    2. Loading the scan

  2. If loading a DICOM file go to File > DICOM. You will be given the option to change the default directory on your system for storing the DICOM database. The "DICOM Browser" window should open.
  3. Click "Import" at the top left of the DICOM Browser.
  4. Locate a folder containing a series/stack of dcm files, select this folder and click "Import". This should add a line to the DICOM Browser database.
  5. To load the scan into the current session select its entry in the database and click "Load" in the bottom left corner of the DICOM Browser window.

    6. Adjusting the display

  6. In the top toolbar click on the "Modules" drop-down menu and select "Volumes".
  7. Open the "Display" tab.
  8. Click on the drop-down menu to the right of "Active Volume" and click on "Rename current Volume" to give the volume a new name (e.g. "Whole body").
  9. Select a preset "Window/Level". For segmenting bones from a CT scan the "CT-bone" option (far left) works well.
  10. Make manual adjustments to the "Window/Level" ("W/L") or "Threshold" as needed using the sliders at the bottom of the Display tab.

    11. Add a new segment

  11. In the top toolbar click on the "Modules" drop-down menu and select "Segment Editor".
  12. In the Segment Editor click "Add".
  13. Set an initial segment using thresholding. Click on the "Threshold" button in the "Effects" section.
  14. Use the slider and numeric input box to set the lower and upper "Threshold Range". Note that colored shading in the 2D viewer panels will update as you change the thresholds. Use this to make sure that everything you want to include in the segmented structure is highlighted. A lower threshold value between 100 and 300 should work well for bone.
  15. Once you've found good threshold levels click "Apply".
  16. To view the segment in 3D click the "Show 3D" button under the "Master volume" drop-down menu.
