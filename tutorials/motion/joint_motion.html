Simulating joint motion

This tutorial will show you how to simulate joint motion. The uninterrupted block of code can be found at the end of this tutorial.

Preliminary steps

All of the code in this tutorial is in the R language so you'll need R installed on your system (you can find R installation instructions here). You will also need to install the beta version (v1.2) of the R package linkR. You can do this by downloading this folder (< 1MB) and then loading the functions into your current R environment using the two commands below.

# Set the path to where the package was downloaded (customize this to your system)
pkg_path <- '/Users/aaron/Downloads/linkR-master/R'

# Load all functions in this folder into the global environment
source_apply <- sapply(paste0(pkg_path, '/', list.files(paste0(pkg_path, ''))), source, .GlobalEnv)
